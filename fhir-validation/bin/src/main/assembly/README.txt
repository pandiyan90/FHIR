This bundle includes snapshots for the following profiles:

- Profiles defined in the base FHIR Specification (v4.0.1: R4 - Mixed Normative and STU) http://hl7.org/fhir/profilelist.html
- US Core Implementation Guide (v3.1.0: STU3 Update) https://www.hl7.org/fhir/us/core/index.html
- CARIN Blue Button Implementation Guide (v0.1.0: STU1 Ballot 1) http://hl7.org/fhir/us/carin-bb/2020Feb/
- HL7 FHIR Implementation Guide: minimal Common Oncology Data Elements (mCODE) Release 1 - US Realm | STU1 (v1.0.0: STU 1) http://hl7.org/fhir/us/mcode/STU1/
- DaVinci Payer Data Exchange (PDEX) Plan Net Implementation Guide (v0.1.0: STU1 Ballot 1) http://hl7.org/fhir/us/davinci-pdex-plan-net/2020Feb/index.html

For more current profiles, one may have to update their environment with the most recent profile.
